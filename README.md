# Force Intellect

This project is a React + Vite landing page for **Force Intellect**, focused on ERP solutions for manufacturing SMEs.

## Features
- Modern responsive homepage layout
- Sticky header and navigation experience
- Theme toggle support
- Sections for products, industries, clients, testimonials, and contact

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS

## Getting Started

### Install dependencies
```bash
npm install
```

### Run locally
```bash
npm run dev
```

### Build for production
```bash
npm run build
```

## Project Structure
- `src/components` — reusable UI and layout components
- `src/pages` — page-level components
- `src/constants` — content and navigation data
- `src/hooks` — custom React hooks

## Deployment
The project can be deployed on platforms such as Vercel, Netlify, or any static hosting provider.
